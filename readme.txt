# SEO Upload & Verification Guide

## Step 1: Upload Files
Upload the following to your GitHub repo root folder:
- sitemap.xml
- robots.txt

## Step 2: Verify Sitemap
Go to Google Search Console → 'Sitemaps' section.
Paste this URL and submit:
https://utimohsin.github.io/AADHAAR-SEWA-KENDRA/sitemap.xml

## Step 3: Add Meta Tags
Paste the meta tags from 'meta-tags-guide.txt' inside the <head> section of all your HTML pages.

✅ Done! Google will start indexing your site soon.
