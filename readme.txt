ASK-GZB.zip — Upload instructions (Hindi + English)

Hindi:
1) GitHub पर जाकर अपने repo में "Add file" → "Upload files" पर क्लिक करें.
2) इस ZIP की फाइलें (index.html, appointment.html, thank-you.html, contact.html, style.css, favicon.ico, hero-illustration.svg, readme.txt) अपलोड करें.
3) Commit changes दबाएं (message: "Deploy site — ASK-GZB").
4) Repo settings → Pages (or "Pages" tab) पर जाएं। Branch चुने (gh-pages या main) और / (root) सिलेक्ट करके Save करें.
5) कुछ मिनट में आपकी साइट: https://USERNAME.github.io/REPO_NAME/ पर live होगी.
6) Google Search में त्वरित indexing के लिए: Google Search Console में अपनी साइट verify करके sitemap submit करें.

English:
1) Go to your repository on GitHub, click "Add file" → "Upload files".
2) Upload all files from this ZIP (index.html, appointment.html, thank-you.html, contact.html, style.css, favicon.ico, hero-illustration.svg, readme.txt).
3) Commit changes (message: "Deploy site — ASK-GZB").
4) Open Settings → Pages, choose branch (main or gh-pages) and root (/) then Save.
5) Your site will be live at https://USERNAME.github.io/REPO_NAME/ within a few minutes.
6) To speed up indexing, verify site in Google Search Console and submit a sitemap.
